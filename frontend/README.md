# LocalLife Agent 前端

基于 Next.js + Tailwind CSS 构建的本地生活规划助手前端界面。

## 技术栈

- **框架**：Next.js 14
- **样式**：Tailwind CSS
- **语言**：TypeScript
- **UI 组件**：计划使用 shadcn/ui

## 快速启动

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器默认运行在 `http://localhost:3000`。

## 页面规划

- 自然语言输入区
- Agent 回复区
- 工具调用日志区
- 方案卡片区
- 确认并安排按钮
- 执行结果区
- 可复制转发消息
