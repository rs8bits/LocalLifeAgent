# LocalLife Agent 前端

基于 Next.js + TypeScript + Tailwind CSS 构建的本地生活规划助手 Demo。前端负责自然语言输入、SSE 流式过程展示、候选方案卡片、多轮修改基准选择、确认执行和转发消息展示。

## 技术栈

- **框架**：Next.js 14
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **接口**：FastAPI Agent API + `fetch()` 读取 SSE 流

## 快速启动

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器默认运行在 `http://localhost:3000`，后端默认读取 `http://127.0.0.1:8000`。

## 已实现页面能力

- 自然语言输入与示例需求。
- 实时过程区：展示输入安全、记忆、改写、意图、工具调用、方案组合、反思和风控事件。
- 方案卡片：展示识别人数、推荐分、时间线、主活动、额外活动、餐饮、饮品、配送、路线、预算、风险提示、推荐理由和团购券。
- 多轮修改：可把某个方案设为“当前修改基准”，再追加“带上我老婆”“下午桌游，晚饭后喝酒”等修改需求。
- 确认执行：用户点击确认后才触发 Mock 预约、订位、团购券订单和配送订单。
- 执行结果：展示执行状态、订单信息和可复制转发消息。
